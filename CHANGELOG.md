# Changelog

All notable changes to this fork (HAPI Cursor Edition) are recorded here. Format groups changes by milestone, then by phase within the milestone, in reverse chronological order.

## Milestone 1 — Refactor & Slim-Down (2026-05-23)

Twelve phases / 60 plans took the fork from upstream multi-agent HAPI to a Cursor-only, single-user-Tailscale codebase with push-gated CI and 0 circular dependencies across `cli/`, `hub/`, and `web/`.

- **Phase 1 — Cut non-Cursor agents:** removed `cli/src/{claude,codex,gemini,opencode}/` + `cli/src/agent/backends/` + Claude `hookForwarder` + Codex GitHub Actions; collapsed `@anthropic-ai/*` deps; ripgrep guard scaffolded.
- **Phase 2 — Cut external integration channels:** removed Telegram bot (`hub/src/telegram/`, `telegramInitData.ts`, `routes/bind.ts`, `grammy`), voice route + ElevenLabs SDK (`hub/src/web/routes/voice.ts`, `web/src/realtime/`, `@elevenlabs/react`), ServerChan push (`hub/src/serverchan/`). `/api/auth` collapsed to access-token-only.
- **Phase 3 — Cut multi-user namespace isolation:** opaque `CLI_API_TOKEN` (no `:ns` split); removed `socket.data.namespace`, JWT `ns` field, namespace-scoped store/cache; SQLite schema v10 drops `users.platform`; offline v9→v10 migration script.
- **Phase 4 — Cut deployment infrastructure:** deleted `hub/src/tunnel/`, `hub/tools/tunwg/`, `hub/scripts/download-tunwg.ts`, `web/src/lib/relay-mode*`; removed `HAPI_RELAY_*` reads + remote log upload; `HAPI_PUBLIC_URL` retained as Tailscale-friendly output.
- **Phase 5 — Flavor consolidation + capability abstraction:** `AgentFlavor` narrowed to `'cursor'`; `shared/src/flavors.ts` capability table populated; ~3200 LoC of multi-flavor branches removed across cli + hub + web; Phase-5 guard block + line-anchored post-filters added.
- **Phase 6 — Agent runtime shared kit + mode hardening:** extracted `cli/src/cursor/modes.ts` + `cli/src/agent/modeConfig.ts` (broke 3 `loop ↔ session ↔ launcher` madge cycles); `UnknownPermissionModeError`; both launchers converged on `permissionModeToCursorArgs`; mid-session yolo + plan switch tests.
- **Phase 7 — Wire contracts unification & SSE patch contract:** `shared/src/schemas.ts` is the single source of `Session / Machine / Message / RunnerState` + `SyncEventSchema`; deleted `hasUnknownSessionPatchKeys()` heuristic; useSSE consumes the canonical schema; broadcast contract tested in hub.
- **Phase 8 — Hub internal decoupling:** `SessionCache` split into `sessionRepository / Liveness / Config / Merge`; `SyncEngine` decomposed into 4 sub-facades; route-helper middleware + `ApiRouteError`; sessions.ts → `routes/sessions/{lifecycle,config,upload,read,index}.ts`; central `KeepaliveScheduler`; D-143 guard block + `check-no-circular-hub.sh`.
- **Phase 9 — Web internal decoupling:** ToolCard 11-file cycle broken; `SessionList.tsx`, `message-window-store.ts`, `settings/index.tsx`, `HappyComposer.tsx`, `_results.tsx` decomposed to under-budget orchestrators; shared utility dedup (`estimateBase64Bytes`, `levenshteinDistance`, `createApiQuery`); D-158 guard block + `check-no-circular-web.sh`.
- **Phase 10 — Config cleanup:** `serverUrl` / `webapp(Url|Host|Origin)` aliases removed; `hapi server` command alias removed; runtime SQLite migrations dropped (schema-version mismatch → reject with offline-tool hint); `loadConfig()` returns frozen value; `_setApiUrl()` / `_setCliApiToken()` / `_setExtraHeaders()` setters deleted; DI cutover across ~30 CLI + 7 Hub consumers.
- **Phase 11 — Test gap fill:** REFT-01 Cursor permission contract matrix (`cli/src/agent/permissionMatrix.test.ts`); REFT-02 SSE reconnect convergence (`useSSE` backoff constants + bounded-window tests, no MAX_RETRIES); REFT-03 auth route + middleware negative cases (16 tests, `assertNoSecretLeak` helper); REFT guard block; Phase 11 coverage captured + non-regression GREEN.
- **Phase 12 — Docs cleanup & milestone verification:** removed `website/` (50 files) + `docs/` (24 files) + `refactor.md`; rewrote root README, AGENTS.md (49 lines), cli/hub/web READMEs from zero (364 lines total); Phase-12 guard block; root `madge:check` script; push + PR CI gate `.github/workflows/verify.yml` (bun 1.3.14 pinned); full automated gate suite green; coverage held vs Phase 11; ROADMAP reconciled against commit history; manual Tailscale + Cursor + phone end-to-end scenario PASS on `e492044`.
