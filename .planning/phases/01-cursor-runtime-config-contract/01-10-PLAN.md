<objective>
Close UAT Test 4: session-list indicators must represent real turn activity and unread completion state, not runner keepalive liveness.

Purpose: Cursor sessions stay active while idle between prompts. The current summary logic treats active=true as running, so blank sessions spin forever. Normal turn completion sends a ready event and updates activity, but no completion marker reaches the Web summary cache. This plan separates idle keepalive from busy work and turns the existing ready event into the unread completed marker UAT expects.

Output: Shared status semantics, Hub status-marker patch propagation, Web SSE convergence fixes, and regression tests across shared, hub, and web.
</objective>
