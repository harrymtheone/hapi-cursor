<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Cursor CLI process to HAPI CLI | Local subprocess output is untrusted and may include noisy stderr. |
| HAPI CLI to Hub RPC | Machine RPC response crosses Socket.IO and must be schema-validated. |
| User-selected model ids to process args | Discovered ids later become launch arguments and must remain data, not shell text. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-04 | Tampering / Elevation | `discoverCursorModels` subprocess invocation | mitigate | Use `spawn` with an argument array and no shell-concatenated user input. |
| T-01-05 | Information Disclosure | discovery failure handling | mitigate | Log stderr locally only; return safe reason categories to Hub/Web. |
| T-01-06 | Denial of Service | model discovery command | mitigate | Enforce a 5000ms timeout and kill the child process on timeout. |
| T-01-SC | Tampering | package installs | accept | No package-manager installs in this plan; package legitimacy audit says no install. |
</threat_model>
