## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Web model route to Hub sync engine | User-selected model change attempts enter server-side session control. |
| Hub sync engine to active CLI session RPC | Runtime acknowledgement determines whether metadata can be persisted. |
| Hub session update to SSE | Persisted config emits strict patches to Web caches. |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-01-14 | Repudiation | model switch UI/backend contract | mitigate | Return explicit status values so UI cannot claim unsupported hot switch success. |
| T-01-15 | Tampering | `POST /api/sessions/:id/model` body | mitigate | Keep Zod body parsing and nullable string contract; route through active-session guard. |
