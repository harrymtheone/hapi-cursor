    - JSON.parse failure on a message → return silently (existing behavior preserved).
    - `SyncEventSchema.safeParse(parsed)` failure → `console.error('[useSSE] dropped malformed event', result.error)` and return; NO call to any invalidate / refetch helper (D-120). The invalidation helpers no longer exist.

    - `session-updated` event → branch on `'metadata' in event.data`: full Session path = setQueryData(session) + upsertSessionSummary; SessionPatch path = patchSessionDetail + patchSessionSummary.
    - `machine-updated` event → branch on `data === null` (removeMachine), `'id' in data` (upsertMachine — full Machine), else (MachinePatchSchema — call removeMachine for the inactive-machine semantic).
    - `session-removed`, `messages-consumed`, `messages-invalidated`, `message-received`, `message-cancelled`, `session-ended`, `connection-*`, `heartbeat`, `toast` → existing branch behavior preserved.
